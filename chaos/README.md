# Chaos

Scénarios d'injection temporelle pour laboratoire local.

- `controlled_clock_demo.py` : rollback déterministe d'une horloge murale simulée.
- `faketime_demo.sh` : démonstration `libfaketime` pour illustrer un processus qui perçoit une autre ligne de temps.

Utilisez ces scripts pour provoquer des comportements reproductibles en test,
pas pour prouver l'exactitude d'un environnement réel.
